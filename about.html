About this Website

Playing with HTML is a tutorial website created by Irene Smith. The site includes tutorials for those wishing to learn web design.

Categories include:

  • HTML
  • CSS
  • JavaScript
  • Graphics

If you don't know any web technologies, start with Your First Page and then go on from there.

Irene Smith is a writer, designer, and programmer. She has been programming since 1982 and started writing about programming and other technical subjects in 1989.

What started out as a hobby has become a career. As of 2019, Irene has worked for many well-known Tech companies including Microsoft, Google, and Mozilla. She is currently the Firefox Developer Content Manager for MDN Web docs, writing about Firefox Developer Tools and Web Extensions.

Irene became a web developer in 1996, when her son was an infant, and once again a hobby became a career. Web Development started as a fun thing to do. She created websites at such places as GeoCities, Tripod, Fortune City, and more.
