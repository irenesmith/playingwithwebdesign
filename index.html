Playing with web design?

Why not?

Sure, there's a lot to learn if you're just starting out, but it's only difficult if you try to learn it all at once. As Mary Poppins once said:

For every job that must be done, there is an element of fun. Find the fun and, snap, the job's a game!

So let's get started!

If you know absolutely nothing about web design, start by reading "Getting Ready for Web Design" where I will help you install the applications you need to have. Then go read "Your First Web Page" and we can get this show on the road. Once you've gotten through the basics, you can begin to make your websites look better by getting started with CSS.
